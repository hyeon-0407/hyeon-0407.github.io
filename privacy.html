Privacy Policy — Auto BGM

This application is a personal automation tool used solely by the developer for managing their own YouTube channel.

Data Collection

This application does not collect, store, or share any user data. It only accesses YouTube API on behalf of the developer's own account to upload videos created by the developer.

The only data accessed is:
